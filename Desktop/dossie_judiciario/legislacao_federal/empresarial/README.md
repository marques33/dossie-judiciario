# Direito Empresarial — Legislação Federal Brasileira

## Descrição

Este diretório contém metadados estruturados das principais **leis de direito empresarial**, incluindo societário, falências, concorrência, contratos empresariais e licitações.

## Arquivo

| `leis_empresariais.jsonl` | 10 | Principais leis empresariais extravagantes |
