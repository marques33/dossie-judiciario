# Enunciados das Jornadas de Direito Comercial (CJF/STJ)

## Ultima atualizacao: Marco de 2026

## Visao Geral

As Jornadas de Direito Comercial sao promovidas pelo Centro de Estudos Judiciarios do Conselho da Justica Federal (CJF), em parceria com o STJ. Reune magistrados, advogados, professores e membros do MP para aprovar enunciados interpretativos sobre direito empresarial.

## Formato dos Registros (JSONL)

```json
  "jornada": 1,

}
```
