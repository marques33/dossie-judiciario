# TRT5 - Tribunal Regional do Trabalho da 5ª Região (Bahia)

## Informações Gerais

- **Sede:** Salvador/BA
- **Jurisdição:** Estado da Bahia
- **Número de desembargadores:** 29
- **Turmas:** 5 turmas (5 desembargadores cada) + 3 Subseções Especializadas
- **Órgão Especial:** 11 desembargadores (presidente + 5 eleitos + 5 mais antigos)
