## Observações

- O TRT-8 tem jurisdição sobre dois estados: Pará e Amapá.
- Cada turma é composta por 5 desembargadores, ordenados por antiguidade.
- Além dos desembargadores nas turmas, 3 desembargadores exercem funções administrativas (Presidência, Vice-Presidência e Corregedoria).
- Em 2025, o tribunal realizou lista tríplice para promoção ao cargo de desembargador.
